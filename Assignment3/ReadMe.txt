Cubic Bezier Spline Tool

This program is a limited pen tool for creating and manipulating a cubic Bezier spline. The user can add points by
clicking on the window and connect them together to form a spline. The program also allows the user to move around
existing nodes and control points to modify the spline.

The program window will appear, and the user can interact with it using the mouse.

To add a new node, click on an empty space within the window.
To move an existing node, click and hold on the node, then drag it around the window.
To move a control point, click and hold on the control point, then drag it around the window.
